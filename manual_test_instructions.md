### 手动测试验证点：如何在游戏中验证该遗物是否生效

1. **准备阶段**：
   - 启动游戏，开始一局新的对战（在主界面选择或直接运行主场景）。
   - 注意观察左上角或核心上方的 UI 显示，记下当前的“核心最大生命值”和“当前生命值”（初始通常为 500/500）。

2. **获取遗物**：
   - 如果游戏内支持遗物获取界面（如波次结算后的奖励选择、或使用调试面板获取遗物），请获取名为“**生命核心 (❤️)**”的遗物。
   - 也可以在 Godot 编辑器运行时，在调试控制台中调用 `GameManager.reward_manager.add_reward("life_core")` 强制获得。

3. **验证数值变化**：
   - 观察控制台输出日志，应出现 `[RewardManager] 生命核心生效: 最大HP +200`。
   - 观察游戏内 UI 显示的生命值：
     - **核心最大生命值**应该增加 200（例如：由 500 变为 700）。
     - **核心当前生命值**也应该同步增加 200（例如：由满血 500/500 变为 700/700）。
   - 如果之前核心受损（如：400/500），获取后应该变为 600/700。

4. **确认效果**：
   - 在接下来受击时，核心血量条上限应当体现为扩充后的数值。
