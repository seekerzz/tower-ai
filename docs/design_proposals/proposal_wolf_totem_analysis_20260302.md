## 3. 关键问题分析

### 3.1 单位购买异常（系统性问题）

**现象**: AI意图购买sheep_spirit，实际购买shell

```
[ACTION] 购买羊灵 (sheep_spirit) - 克隆核心
[GAME] 【单位购买】购买了 shell，放入了 bench 坐标 0
```

**跨图腾对比**:

| 图腾测试 | 意图购买 | 实际购买 |
|----------|----------|----------|
| 牛图腾 | iron_turtle | bear/plant/yak_guardian |
| 蝙蝠图腾 | mosquito | bear |
| 狼图腾 | sheep_spirit | shell |

**结论**: 这是**系统性问题**，影响所有图腾测试。可能原因：
1. 商店显示与实际库存不一致
2. 购买API的参数解析存在bug
3. 单位名称到ID的映射错误
