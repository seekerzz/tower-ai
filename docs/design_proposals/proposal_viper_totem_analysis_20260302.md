## 3. 关键问题分析

### 3.1 商店阵营过滤异常

**现象**: 选择毒蛇图腾后，商店显示非毒蛇阵营单位

**可能原因**:
1. 阵营过滤逻辑未正确实现
2. 单位阵营标签缺失或错误
3. 过滤条件过于宽松（如仅降低概率而非完全过滤）

**对比其他图腾测试**:
- 牛图腾测试：商店显示yak_guardian/rage_bear等牛阵营单位 ✅
- 蝙蝠图腾测试：商店显示torch/cow_golem/sunflower等（不太明显）⚠️
- 狼图腾测试：商店显示shell/cauldron/dog等（不太明显）⚠️
- 毒蛇图腾测试：显示lucky_cat/squirrel等非阵营单位 ❌

### 3.2 单位购买异常

**现象**: AI意图购买毒蛇阵营单位，实际购买lucky_cat
