# 测试任务：鹰图腾核心机制验证

## 测试目标
验证鹰图腾的暴击回响机制：暴击时有 30% 概率触发一次回响，造成等额伤害并施加攻击特效。

## 测试等级覆盖
- [x] Lv.1 必须测试（验证基础机制）
- [ ] Lv.2/Lv.3 仅测试机制变化（纯数值提升无需测试）

## 关联机制
- Buff: 暴击 (crit) - 状态：✅ 已校验
- Debuff: 无
- 图腾技能：暴击回响

## 图腾分类确认
| 确认项 | 内容 | 来源 |
|--------|------|------|
| 图腾 ID | eagle_totem | data/units/eagle_totem.json |
| 所属阵营 | 鹰图腾 | GameDesign.md |

## 前置条件
- 图腾：鹰图腾 (eagle_totem)
- 单位：1 个老鹰 (eagle) 或任意鹰图腾单位
- 敌人：1 个普通敌人（用于观察伤害）
- 作弊 API 调用：`set_core_hp(99999)`, `set_unit_crit(100)` (可选，提高暴击率)

## 验证点
| 验证项 | 预期日志/现象 | 验证方式 |
|--------|--------------|----------|
| 暴击触发 | 单位攻击产生暴击 | 日志分析 |
| 回响触发 | `[TOTEM_EFFECT] 暴击回响触发，额外伤害：{值}` | 日志分析 |
| 回响伤害 | 回响造成等额伤害 | 伤害数值分析 |
| 攻击特效 | 回响触发时施加攻击特效 | 日志分析/行为观察 |
| 触发概率 | 约 30% 暴击触发回响 | 统计分析 |

## 场景构造步骤
1. 选择鹰图腾开局
2. 调用作弊 API `set_core_hp(99999)` 构造高血量核心
3. 购买并放置鹰图腾单位
4. 召唤 1 个敌人用于观察
5. （可选）调用作弊 API 提高暴击率
6. 观察攻击和暴击回响触发情况，记录日志
7. 验证回响伤害是否正确

## 依赖排除项（已校验，无需重复测试）
- 暴击 Buff: 已在其他测试中校验

## 备注

- 注意观察回响是否造成等额伤害并施加攻击特效
