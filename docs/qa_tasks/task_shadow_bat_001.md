# 测试任务：影蝠 (ShadowBat) 单位验证

## 测试目标
验证影蝠单位的暗影步瞬移机制和流血效果

## 测试等级覆盖
- [x] Lv.1 必须测试（验证基础机制：瞬移 + 流血 + 返回）
- [ ] Lv.2 轨迹减速（机制变化，需要额外测试）
- [ ] Lv.3 轨迹伤害（机制变化，需要额外测试）

## 关联机制
- Buff: 无
- Debuff: 流血 (Bleed) - 状态：✅ 已校验
- 单位技能：暗影步、瞬移
- 被动效果：冷却时间 6 秒

## 前置条件
- 图腾：蝙蝠图腾 (bat_totem)
- 单位：1 个影蝠 (shadow_bat)
- 敌人：多个普通敌人（分布在不同位置）
- 作弊 API 调用：
  - `set_god_mode(true)` - 保护核心
  - `spawn_enemy` - 在指定位置生成敌人

## 验证点
| 验证项 | 预期日志/现象 | 验证方式 |
|--------|--------------|----------|
| 暗影步瞬移 | 每 6 秒瞬移到敌人身边 | 日志分析 + 行为观察 |
| 选择最远敌人 | 优先瞬移到距离最远的敌人 | 行为观察 |
| 落点流血 | 落点周围敌人获得 1 层流血 | 日志分析 |
| 返回原位 | 瞬移后返回原位置 | 行为观察 |
| 冷却时间 | 两次瞬移间隔约 6 秒 | 时间统计 |

## 场景构造步骤
1. 选择蝙蝠图腾
2. 购买影蝠并放置到战场中心 (0,0)
3. 开启上帝模式保护核心
4. 开始第 1 波生成敌人，或手动生成多个敌人在不同位置
5. 观察并记录：
   - 影蝠瞬移动作和日志
   - 瞬移目标是否为最远敌人
   - 落点周围敌人是否获得流血
   - 瞬移后是否返回原位
   - 记录两次瞬移的时间间隔
6. 统计验证：
   - 总攻击次数和瞬移触发次数
   - 冷却时间是否在 5.5-6.5 秒范围内
