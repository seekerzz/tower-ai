# 测试任务：鹰图腾核心机制验证（复测）

## 测试目标
验证鹰图腾的暴击回响机制：
- 暴击时有 30% 概率触发一次回响
- 造成等额伤害并施加攻击特效

## 前置条件
- 图腾：鹰图腾 (eagle_totem)
- 单位：1 个老鹰 (eagle) 或任意鹰图腾阵营单位（推荐角雕 harpy_eagle，第 3 次攻击必定暴击）
- 敌人：1 个以上普通敌人

## 场景构造
1. 选择鹰图腾开局
2. 购买角雕 (harpy_eagle) - 第 3 次攻击必定暴击
3. 将单位部署到战场坐标 **(-1, 0)** 或 **(1, 0)**（⚠️ 禁止使用 (0,0) 核心位置）
4. 开始波次，让敌人进入战场
5. 观察 30 秒战斗日志

## 预期验证点

| 验证项 | 预期日志 | 验证方式 |
|--------|---------|---------|
| 图腾选择 | 鹰图腾选择成功 | 系统日志 |
| 暴击触发 | `造成暴击，伤害：{值}` | 日志分析 |
| 回响触发 | `[TOTEM_EFFECT] 鹰图腾 触发暴击回响` | 日志分析 |
| 回响伤害 | `回响伤害：{值}` | 日志分析 |
| 攻击特效 | 回响施加攻击特效 | 日志分析 |

## 注意事项
⚠️ **重要**：单位部署坐标不能是 (0, 0)，该位置是核心所在位置
- 推荐坐标：(-1, 0), (1, 0), (-1, -1), (1, 1) 等

## 依赖排除项
- 暴击机制：已在其他测试中校验
- 鹰图腾单位攻击机制：部分已校验
