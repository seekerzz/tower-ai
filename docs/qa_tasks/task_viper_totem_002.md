# 测试任务：毒蛇图腾核心机制验证（复测）

## 测试目标
验证毒蛇图腾的毒液攻击机制：
- 每 5 秒对距离最远的 3 个敌人降下毒液
- 造成伤害并施加 3 层中毒

## 前置条件
- 图腾：毒蛇图腾 (viper_totem)
- 单位：1 个毒蛇 (viper) 或任意毒蛇图腾阵营单位
- 敌人：3 个以上普通敌人

## 场景构造
1. 选择毒蛇图腾开局
2. 购买任意毒蛇阵营单位
3. 将单位部署到战场坐标 (-1, 0) 或 (1, 0)（避开核心位置）
4. 开始波次，让敌人进入战场
5. 观察 60 秒战斗日志

## 预期验证点

| 验证项 | 预期日志 | 验证方式 |
|--------|---------|---------|
| 图腾攻击触发 | `[TOTEM] 毒蛇图腾触发毒液攻击，目标：[enemy1, enemy2, enemy3]` | 日志分析 |
| 毒液伤害 | `[TOTEM_DAMAGE] 毒液对敌人{ID}造成{伤害值}伤害` | 日志分析 |
| 中毒层数 | `[TOTEM_EFFECT] 敌人{ID}中毒层数：3` | 日志分析 |
| 目标选择 | 选择距离最远的 3 个敌人 | 日志分析 |
| 触发频率 | 约每 5 秒触发一次 | 时间戳分析 |

## 备注
- 如果 HTTP 连接超时，尝试缩短观察时长至 30 秒
- 确保单位部署坐标不是 (0,0) 核心位置
- 建议部署 2-3 个单位以确保输出足够

## 依赖排除项
- 中毒 Debuff: ✅ 已校验（在之前测试中已验证）
