# 测试任务：石像鬼 (Gargoyle) 单位验证

## 测试目标
验证石像鬼单位的石化机制和伤害反弹效果

## 测试等级覆盖
- [x] Lv.1 必须测试（验证基础机制：石化 +20% 反弹）
- [ ] Lv.2 反弹 2 次（数值提升，无需重复测试机制）
- [ ] Lv.3 反弹 3 次（数值提升，无需重复测试机制）

## 关联机制
- Buff: 无
- Debuff: 无
- 单位技能：石化形态、伤害反弹
- 被动效果：核心血量阈值触发

## 前置条件
- 图腾：蝙蝠图腾 (bat_totem)
- 单位：1 个石像鬼 (gargoyle)
- 敌人：基础攻击型敌人
- 作弊 API 调用：
  - `set_god_mode(true)` - 保护核心
  - `set_core_hp(hp)` - 设置核心血量触发石化

## 验证点
| 验证项 | 预期日志/现象 | 验证方式 |
|--------|--------------|----------|
| HP<30% 石化 | 核心血量低于 30% 时进入石像形态 | 日志分析 + 行为观察 |
| 石化停止攻击 | 石像形态下不主动攻击 | 行为观察 |
| 反弹 20% 伤害 | 攻击者受到 20% 反弹伤害 | 日志分析 |
| HP>60% 恢复 | 核心血量高于 60% 时恢复常态 | 日志分析 + 行为观察 |

## 场景构造步骤
1. 选择蝙蝠图腾
2. 购买石像鬼并放置到战场
3. 开启上帝模式保护核心
4. 设置核心血量为 20%（如 100/500）触发石化
5. 开始波次生成敌人
6. 观察并记录：
   - 石化状态变化日志
   - 敌人攻击时的反弹伤害
   - 石像鬼是否停止攻击
7. 设置核心血量为 80%（如 400/500）验证恢复
8. 观察并记录恢复常态日志

## 预期日志
```
[TOTEM] 石像鬼 进入石化形态 - 核心 HP 低于 30%
[REFLECT] 敌人{ID} 受到反弹伤害 {伤害值} (20%)
[TOTEM] 石像鬼 恢复常态 - 核心 HP 高于 60%
```

## 备注
- 重点验证石化阈值 (30%/60%) 是否正确
- 验证反弹伤害比例是否为 20%
- 如有日志埋点不足，需上报技术总监补充
