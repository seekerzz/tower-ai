# 测试任务：影蝠暗影步机制验证

## 测试目标
验证影蝠 (ShadowBat) 的暗影步机制：
1. 每 6 秒瞬移到最远敌人身边
2. 瞬移后返回原位
3. 冷却时间 6 秒

## 测试等级覆盖
- [x] Lv.1 必须测试（验证基础机制）
- [x] Lv.2: 冷却 6 秒→4 秒，暗影轨迹 - 需要测试机制变化
- [x] Lv.3: 暗影轨迹持续 3 秒，轨迹伤害 - 需要测试机制变化

## 关联机制
- Buff: 无
- Debuff: 流血 (Bleed) - 状态：✅ 已校验
- 单位技能：暗影步

## 前置条件
- 图腾：蝙蝠图腾
- 单位：1 个影蝠 (shadow_bat)
- 敌人：2 个普通敌人（一个近，一个远）
- 作弊 API 调用：`set_core_hp(99999)`, `spawn_enemy(2)`

## 验证点
| 验证项 | 预期日志/现象 | 验证方式 |
|--------|--------------|----------|
| 瞬移触发 | 影蝠瞬移到最远敌人身边 | 行为观察 |
| 目标选择 | 选择距离最远的敌人 | 行为观察 |
| 落点流血 | 落点周围敌人获得 1 层流血 | 日志分析 |
| 返回原位 | 瞬移后返回原位 | 行为观察 |
| 冷却时间 | 6 秒 cooldown | 时间间隔观察 |

## 场景构造步骤
1. 选择蝙蝠图腾开局
2. 调用作弊 API `set_core_hp(99999)` 构造高血量核心
3. 购买并放置影蝠在固定位置
4. 召唤 2 个敌人，一个近一个远
5. 观察影蝠是否瞬移到最远敌人身边
6. 观察影蝠是否返回原位
7. 记录瞬移间隔时间，验证冷却时间

- 流血 Debuff: 已在其他测试中校验
