# 测试任务：石像鬼反弹伤害验证

## 测试目标
验证石像鬼 (Gargoyle) 在石像形态下反弹 20% 伤害给攻击者的机制。

## 测试等级覆盖
- [x] Lv.1 必须测试（验证基础机制）
- [ ] Lv.2: 反弹 2 次 - 需要测试机制变化
- [ ] Lv.3: 反弹 3 次 - 需要测试机制变化

## 关联机制
- Buff: 石化 (Petrify) - 状态：✅ 已校验
- Debuff: 无
- 单位技能：石化反弹

## 前置条件
- 图腾：蝙蝠图腾
- 单位：1 个石像鬼 (gargoyle)
- 敌人：1 个普通敌人（攻击力已知，如攻击力=100）
- 作弊 API 调用：`set_core_hp(100)`, `set_unit_hp(gargoyle, 50)` (使石像鬼 HP<30%)

## 验证点
| 验证项 | 预期日志/现象 | 验证方式 |
|--------|--------------|----------|
| 石化形态 | 石像鬼 HP<30% 时进入石像形态 | 行为观察 |
| 停止攻击 | 石像形态下停止主动攻击 | 行为观察 |
| 反弹伤害 | 敌人受到反弹伤害=攻击力×20% | 日志分析 |
| 反弹次数 | Lv.1 反弹 1 次，Lv.2 反弹 2 次，Lv.3 反弹 3 次 | 日志分析 |

## 场景构造步骤
1. 选择蝙蝠图腾开局
2. 调用作弊 API `set_core_hp(100)` 构造低血量核心
3. 购买并放置石像鬼
4. 调用作弊 API `set_unit_hp(gargoyle, 50)` 使石像鬼 HP<30%（假设满血 200）
5. 召唤 1 个攻击力已知的敌人攻击石像鬼
6. 观察敌人是否受到反弹伤害
7. 记录反弹次数和伤害数值
