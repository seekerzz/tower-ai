# 测试任务：毒蛇图腾核心机制验证 (复测 003)

## 测试目标
验证毒蛇图腾的毒液攻击机制是否正确触发并通过日志输出。

## 测试等级覆盖
- [x] Lv.1 必须测试（验证基础机制）
- [ ] Lv.2/Lv.3 仅测试机制变化（纯数值提升无需测试）

## 关联机制
- Buff: 无
- Debuff: 中毒 (poison)
- 单位技能：毒蛇图腾毒液攻击
- 被动效果：每 5 秒对最远 3 个敌人施加毒液

## 前置条件

- 敌人：基础敌人（用于触发攻击）

## 场景构造步骤
1. 启动游戏客户端，选择毒蛇图腾
2. 刷新商店直到出现箭毒蛙 (arrow_frog)
3. 购买并部署箭毒蛙到坐标 (1, 0)
4. 调用 start_wave 开始波次
5. 观察战斗 60 秒，轮询 /observations 获取日志
6. 检测日志中的 [VIPER_TOTEM] 和 [VIPER_POISON] 标记
